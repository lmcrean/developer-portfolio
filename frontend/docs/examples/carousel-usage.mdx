## How to Use

To use the ProjectCarousel component in your MDX files:

1. Import the component at the top of your MDX file:
   ```jsx
   import ProjectCarousel from '@site/src/components/ProjectCarousel';
   ```

2. Use the component with the appropriate project key:
   ```jsx
   <ProjectCarousel projectKey="odyssey" />
   ```

3. Make sure the project is defined in the `projectCarousels.js` data file.

Note: The SplideInit component is automatically included in your app via the theme, so you don't need to import it in each MDX file.
